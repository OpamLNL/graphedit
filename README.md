# GraphEdit Client

Клієнтська частина проєкту **GraphEdit** — інтерактивна карта знань, яка візуалізує вузли тем, зв'язки між ними та дозволяє взаємодіяти з даними через REST API.

## 🔧 Технології

- [React](https://reactjs.org/)
- [Vite](https://vitejs.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Yarn](https://yarnpkg.com/)
- [vis-network](https://visjs.github.io/vis-network/) (або інша бібліотека графів)
- REST API на NestJS (`/nodes`, `/node-connections`)
